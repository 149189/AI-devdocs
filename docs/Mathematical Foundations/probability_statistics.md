# Probability & Statistics

Provides tools for modeling uncertainty and variability in data.

## Key Concepts
- **Random Variable**: A variable with probabilistic outcomes.
- **Expectation (E[X])**: Average value.
- **Variance / Std Dev**: Measures spread.
- **Bayes' Theorem**: Updates beliefs with new data.
- **Distribution Types**: Normal, Bernoulli, Binomial, Poisson.

## ML Applications
- Bayesian inference.
- Probabilistic models (Naive Bayes, HMMs).
- Confidence intervals & hypothesis testing.
